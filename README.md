# 🏦 Smart Banking System (Flask Web App)

A simple and interactive **Banking Web Application** built using **Python, Flask, and SQLite**.
This project simulates basic banking operations like account creation, login, deposit, withdrawal, and balance checking through a web interface.

---

## 🚀 Features

* 👤 Create New Account
* 🔐 Secure Login System
* 💰 Deposit Money
* 💸 Withdraw Money
* 💳 Check Balance in Real-Time
* 🗄️ Data Stored using SQLite Database
* 🌐 Web-based UI using Flask + Bootstrap

---

## 🛠️ Tech Stack

* **Backend:** Python (Flask)
* **Frontend:** HTML, Bootstrap
* **Database:** SQLite

---

banking_app/

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/your-username/banking-app.git
cd banking-app
```

### 2️⃣ Install Dependencies

```
pip install flask
```

### 3️⃣ Run the Application

```
python app.py
```

## 🌐 Usage

Open your browser and go to:

```
http://127.0.0.1:5000
```

### ✔ Steps:

1. Create a new account
2. Login using account details

   * Deposit
   * Withdraw
   * Check Balance
